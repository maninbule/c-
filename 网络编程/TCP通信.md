# TCP通信与UDP对比
```
// TCP 和 UDP -> 传输层的协议
UDP:用户数据报协议，面向无连接，可以单播，多播，广播， 面向数据报，不可靠
TCP:传输控制协议，面向连接的，可靠的，基于字节流，仅支持单播传输
                   UDP                            TCP
是否创建连接       无连接                         面向连接
是否可靠           不可靠                         可靠的
连接的对象个数     一对一、一对多、多对一、多对多   支持一对一
传输的方式         面向数据报                      面向字节流
首部开销           8个字节                         最少20个字节
适用场景           实时应用（视频会议，直播）       可靠性高的应用（文件传输）
```
